## IPOCS Model Railway

Internet Protocol Object Controller System for Model Railway is a Open Source Software system for managing wayside objects in a model railway using cheap hardware.
It is developed for using with ESP8266 as network node and an Arduino Uno as processing node, connected to eachother using the serial connection pins.
